斗兽棋 for iPad in objective-c

作者：徐豪俊 QQ:66641821

斗兽棋 in objective-c 是一款使用objective-c 开发的小游戏。
目前实现的功能是双人单机对战，暂不支持单人模式。
玩法与通用的斗兽棋的规则一致
